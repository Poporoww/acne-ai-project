# Facial Acne Detection and Consultation App

This project is an Android application that detects types of facial acne and provides personalized skincare advice using artificial intelligence.

## 🔍 Features

- Detects acne from user-uploaded or captured facial image using **YOLOv5**
- Classifies multiple types of acne: blackhead, whitehead, papules, pustules, cysts, and nodules
- Provides skincare suggestions based on detected acne via **Gemini API**
- Saves analysis history using **Firebase Realtime Database**
- Built with **Java** on **Android Studio**
